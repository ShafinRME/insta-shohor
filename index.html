How does JavaScript Work?

JavaScript is a client-side scripting language.That means it runs at the client side inside the web-browsers.But here is the one pre-condition and it is that client browser must be JS enabled.For enabling JS in browser there are many types of Engine.These engine will be vary in browser to browser.In chrome browser the JS engine's name is V8. In a standard JS engine there are 4 steps to execute the source code to machine code. 1.Parser - Receiver. 2.AST(Abstract Syntax Tree) - Create a Data Structure by solving Errors. 3.Conversion to Machine Code - Convert the JS code to Machine code. 4.Machine Code - Previously converted code is sent to the system for execution and Finally byte code run by the system. This is the way how JS works.
